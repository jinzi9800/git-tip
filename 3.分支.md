1.查看当前分支
---  

`git status`

2.产看所有分支
---  

`git branch`

3.创建分支
---  

`git branch【分支名】`

4.切换分支
---  

`git checkout 【分支名】`

5.对比分支
---  

`git diff 【分支名如master】..【另一个分支名】`

6.合并分支
---  
在一个分支中，命令：

`git merge 【要合并分支的另一个名称】`

7.解决冲突的办法
---  

git merge完后，会加标记显示不同，删除其中的一个，保存就ok。

8.重命名和删除分支
---  

`git branch -m 【旧分支名】 【新分支名】`
