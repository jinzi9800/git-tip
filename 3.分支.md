1. `git status`  
查看当前分支  

2. `git branch`  
查看所有分支  

3. `git branch【分支名】`  
创建分支  

4. `git checkout 【分支名】`  
切换分支  

5. `git diff 【分支名如master】【另一个分支名】`  
对比分支  

6. `git merge 【要合并分支的另一个名称】`  
合并分支 在一个分支中 git merge完后，会加标记显示不同，删除其中的一个，保存就ok。  

7. `git branch -m 【旧分支名】 【新分支名】`  
重命名分支  

8. `git branch -d 【分支名】`
删除分支
